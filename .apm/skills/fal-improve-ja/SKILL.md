# 日本語の推敲と監査

このスキルは、書き上がった日本語の文章への推敲・監査のパスです。目標とする状態は fal-write-ja が定めます。監査の前に fal-write-ja の本文を読み、[用例集](./references/examples.md) を通読してパターンを把握してください。対象は、直近の作業で書いた日本語、またはユーザーが指定した文章です。

## 監査者の姿勢

用法の失敗（fal-write-ja §1〜§3）は、生成の土台となるパターンそのものが失敗の源であるため、書いた本人には自然に見えます。可能なら、推敲・監査は書いたセッションとは別の文脈で行います。新しいセッションやサブエージェントに、このスキルと fal-write-ja、対象の文章だけを読ませます。書いたときの文脈を共有しない読み手のほうが、確実に見つけられます。

修正の加減は fal-write-ja §7 に従います。定着した術語、プロジェクトで定義済みの語、ユーザー自身が使っている語まで置き換えるのは過剰修正です。

## 手順

少なくないケースが文法の検査を通ってしまうので、語の単位を中心に見ます。§ の参照は fal-write-ja 本文の節を指します。手順1〜4は、語を拾う手がかりの違いで分かれています。手順5〜6は語ではなく形の点検、手順7は自分の言い直しに対する検査です。以下の手順で問題を見つけ、原則としてその場で修正します。

1. 語の照合: 用例集の「誤用されやすい語の一覧」にある語を、対象の文章から機械的に探します。見つけた語は機械的に置き換えるのではなく、項目の説明で場面を確かめてから言い直します。一覧は頻度の高い失敗を先に拾うための補助であり、照合を終えても、次の用法の確認は省略できません。
2. 用法の確認: 文意を支える語（漢語・専門用語・比喩）ごとに、その語がその意味で、技術文書で一般に使われているかを問います（§1）。用法の確かめられない語は、造語・別の場面からの流用・英語の語義の直訳のいずれかを疑って言い直します（§1〜§3）。比喩や漢語といった語の種類は退ける理由にならず、プロジェクトや分野で定着した術語は、この確認の対象外です（§7）。語の形に応じて、次の点まで問います。
   - 複合語は、まず定着した型に当たるかと、省いた語を補うと戻し方が一通りに決まるかを問います（§2）。定着した型に当たる連結なら、連結であること自体を理由に退けることはできません。あとは構成する各語の用法を確認します。そうではない連結は、連結そのものの用法を問います。
   - 述語は、主語・目的語との組み合わせまで問います。
   - 見出しや総称に使う語は、それが束ねる各項目との関係を文に戻し、項目ごとの組み合わせを問います。項目は主語とは限らず、目的語のこともあります。
3. 逆翻訳: 引っかかる表現を英語に訳し戻し、自然な英語の慣用句が現れたら直訳を疑って言い直します（§3）。
4. 文体の水準: 物差しは、公式リファレンスのような正式な技術文書に違和感なく現れる語彙です。これより難しく専門的な語（学術用語・畑違いの術語・堅い漢語）は、この範囲の語だけで言い直せるかを試し、失うものがないなら言い直した側を採ります。失うものがあるなら、その術語は役割を果たしているので残します。逆に、砕けすぎた語（雑談でしか言わない言い回し）は、文書の語へ言い直します。
5. 構文と修辞の点検: 行為を分岐させない条件形、体言止め、近くで重なる比喩、「X ではなく Y」の多用を探して書き直します（§4〜§5）。
6. 記号の点検: コロン・ダッシュ・強調の鉤括弧を機械的に探し、英語の用法のものを書き直します（§6）。
7. 置き換えの検査: 言い直しの案を元の表現と並べ、ここまでと同じ検査（用法・組み合わせ・文体）を案の側にもかけます。元より良くならないなら、元を残します。どの案も検査を通らないなら、元を失敗とした判定の側を疑います。本当に失敗なら、語を補って説明的に言い直せば、その案は検査を通るはずだからです。それでも元が失敗に見えるなら、語の選択ではなく命名や概念の切り分けの問題です。修正せず、ユーザーに提起します。

## 権限

このスキルは通常、見つけた問題をその場で修正します。執筆作業の仕上げとして実行するときと、修正、推敲、校正、または書き直しを求められたときは、指定された範囲を編集します。ユーザーが編集を伴わないレビュー、監査、評価、または報告を明示的に求めた場合に限り、対象の文章を変更せず、問題点と具体的な修正案を報告します。

## 編集の規律

- 監査を通った文は、字句も改行もそのまま残します。目的のない書き換えは差分のノイズになり、書き直した文には新たな失敗が入り込みます。
- 言い直しは fal-write-ja の規則に全面的に従い、手順7の検査を通った案だけを採ります。
